Css Units

  1. px(pixels)=>fixed =>not used for responsive design
  2. rem => related to the root element => 1rem=16px
  3. em=> relaated to the parent element
  4. % => related to the parent
  5. vh => related to the view points height
  6. vw => related to viewpoint width
